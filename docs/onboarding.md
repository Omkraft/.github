# Developer Onboarding

## What You Need to Know

- You do NOT manage releases
- You do NOT bypass CI
- You MUST follow Conventional Commits

---

## PR Checklist

- Correct title format
- Lint passes
- No force pushes to main

---

## If Something Fails

1. Check Actions tab
2. Read error message
3. Fix commit title or code
4. Re-push

Ask platform team only after that.
